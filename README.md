# KreślBmp, biblioteka graficzna w ANSI C

Jan Mleczko, 2026

KreślBmp 1.0 to biblioteka podprogramów w języku programowania C do kreślenia rysunków i zapisywania ich w formacie BMP. Jest w pełni przenośna polegając tylko na funkcjach standardowych.

Grafika demonstracyjno-testowa:

![](https://github.com/Jan-Mleczko/KreslBmp-library/raw/refs/heads/main/KBmp10demo.bmp)

Pliki w repozytorium:

* `KRESLBMP.C` - kod źródłowy biblioteki do skompilowania do pliku obiektowego,
* `KRESLBMP.H` - plik nagłówkowy,
* `KBMP_DOK.HTM` - pełna **dokumentacja** biblioteki, 
* `KBMP10DEMO.BMP` - powyższa grafika demonstracyjna,
* Czcionki zewnętrzne (o co chodzi - patrz dokumentacja):
  * `SEGM7.CZC` - czcionka z cyframi i minusem inspirowana wyświetlaczami 7-segmentowymi,
  * `ALFAWEKT.CZC` - czcionka wektorowa zawierająca tylko litery polskie bez ogonków, spację oraz "X" i "V" do zapisu liczb rzymskich.

Więcej w [dokumentacji - pobierz ją](https://github.com/Jan-Mleczko/KreslBmp-library/raw/refs/heads/main/kbmp_dok.htm).
